Working with StreamSheets
=========================

There are a million things you can do with the power of a Streamsheet. Any logic you can think of can be implemented with simple or rarely complex functions. A Stream Sheet holds the same and more possibilities as any other Spreadsheet software you have been using in the past.
In this tutorial you will get the chance to understand our UI in more detail and we will give you some basic Use Cases to implement, which will stimulate your creative problem solving skills.
