TAN
-----------------------------

Returns the tangent value of the angle.

**Syntax**

=TAN(Angle)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Angle
     - Angle in radians

**Return Value**

Evaluated tangent value.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =TAN(PI()/4)
     - 1
     -
