ARCCOS
-----------------------------

Calculates the arccosine value of the given value

**Syntax**

=ARCCOS(Value)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value
     - Value to evaluate arccosine for.

**Return Value**

Evaluated angle in radians.

   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =ARCCOS(0.3)
     - 1.26610367
