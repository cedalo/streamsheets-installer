SIGN
-----------------------------

Returns the sign of given number.

**Syntax**

=SIGN(Value)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value
     - Value to check.

**Return Value**

Returns 1, if number value is positive, -1, if it is negative or 0, if value is 0.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =SIGN(-3)
     - -1
     -
