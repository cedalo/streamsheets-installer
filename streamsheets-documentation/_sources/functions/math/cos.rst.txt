COS
-----------------------------

Returns the cosine value of the angle.

**Syntax**

=COS(Angle)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Angle
     - Angle in radians

**Return Value**

Evaluated cosine value.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =COS(PI())
     - -1
