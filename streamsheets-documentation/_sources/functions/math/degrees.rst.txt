DEGREES
-----------------------------

Converts radian value to degrees.

**Syntax**

=DEGREES(Radians)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Radians
     - Radian value to convert.

**Return Value**

Degrees converted from radian angle.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =DEGREES(PI())
     - 180
     -
