ARCSIN
-----------------------------

Calculates the arcsine value of the given value

**Syntax**

=ARCSIN(Value)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value
     - Value to evaluate arcsine for.

**Return Value**

Evaluated angle in radians.

   :widths: 45 10 45
   :header-rows: 1

     - Result
     - Comment
