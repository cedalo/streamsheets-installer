MAX
---

Evaluates the maximum value of all given values.

**Syntax**

=MAX(Value1, Value2...)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value1, Value2...
     - Number values to evaluate max from.

**Return Value**

Maximum value from all given values.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =MAX(3, 6, 12)
     - 12
     -
