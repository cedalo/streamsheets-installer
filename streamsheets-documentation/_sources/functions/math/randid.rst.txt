RANDID
-----------------------------

Creates a random key with the given amount of chars.

**Syntax**

=RANDID(Length)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Length
     - Length of key to generate. The longer the key, the higher the probability of receiving a unique key.

**Return Value**

Random key as String.

**Example**

.. list-table::
   :widths: 30 20 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =RANDID(7)
     - e.g.: Z83hab4
     - The return value can be any combination of characters and numbers.
