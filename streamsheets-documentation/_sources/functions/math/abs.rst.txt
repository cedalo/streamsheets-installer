ABS
---

Returns the absolute value of a number.

**Syntax**

=ABS(Value)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value
     - Value of which to return the absolute number.

**Return Value**

The absolute number.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =ABS(-3)
     - 3
     - Absolute number is three
