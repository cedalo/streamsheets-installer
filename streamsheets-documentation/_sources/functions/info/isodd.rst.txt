ISODD
-----------------------------

Checks, if given value is an odd number.

**Syntax**

=ISODD(Value)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value
     - Value to check.

**Return Value**

TRUE if value is an odd number, otherwise FALSE.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =ISODD(3)
     - TRUE
     - Three is a odd number.
