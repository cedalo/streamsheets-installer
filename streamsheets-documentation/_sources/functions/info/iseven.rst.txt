ISEVEN
-----------------------------

Checks, if given value is an even number.

**Syntax**

=ISEVEN(Value)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value
     - Value to check.

**Return Value**

TRUE if value is an even number, otherwise FALSE.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =ISEVEN(3)
     - FALSE
     - Three is no even value.
