
AVERAGE
-------

Returns the average of specified cell values. Values which do not represent a number are ignored.

**Syntax**

=AVERAGE(Value1, [Value2], ...)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value1
     - Cell or cell-range to calculate average for.
   * - Value2, ...
     - Optional. Additional Cells or cell-ranges to calculate average for.

**Return Value**

The average over all specified cell values.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =AVERAGE(4, 5, 6, 7)
     - 5.5
     -
