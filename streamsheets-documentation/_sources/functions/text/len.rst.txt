LEN
-----------------------------

Counts characters in given text.

**Syntax**

=LEN(Text)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Text
     - Text to count characters of.

**Return Value**

Number of characters in Text.

**Example**

.. list-table::
   :widths: 50 50
   :header-rows: 1

   * - Function
     - Result
   * - =LEN("Somewhere")
     - 9
