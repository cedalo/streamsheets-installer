MID
-----------------------------

Return the given amount of characters starting at the given index from a string.

**Syntax**

=MID(Text, Index, Count)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Text
     - Text to extract string from.
   * - Index
     - Index to start extracting from.
   * - Count
     - Number of characters to extract.

**Return Value**

The text that that is extracted from the given text using the arguments.

**Example**

.. list-table::
   :widths: 50 50
   :header-rows: 1

   * - Function
     - Result
   * - =MID("SampleText", 3, 2)
     - "pl"
