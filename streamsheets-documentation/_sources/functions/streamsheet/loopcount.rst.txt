LOOPCOUNT
-----------------------------

Returns the length of specified loop array or :ref:`#NA <error>` if loop is not available or not defined. Used to analyze :ref:`Inbox` loops.


**Syntax**

=LOOPCOUNT()

**Arguments**

None.

**Return Value**

Number, representing the length of defined loop array.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =LOOPCOUNT()
     - 3
     - If the loop array contains 3 elements
