UDP.SEND
-----------------------------

Publishes either a message or an arbitrary value using a specified :term:`Producer`, host and port. You can use the :ref:`Function Wizard <functionwizard>` for this function. 


**Syntax**

=UDP.PUBLISH(Producer, MessageorValue, Host, Port)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Producer
     - Producer to use for publishing.
   * - MessageorValue
     - Either an existing message from the outbox or a value to publish.
   * - Host
     - 
   * - Port
     - 
   

**Return Value**

TRUE, if successful.
