DELETECELLS
-----------------------------

Delete cell values in given range. Is able to delete itself.

**Syntax**

=DELETECELLS(Range)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Range
     - Range of cells to delete.

**Return Value**

TRUE, if successful.

   :widths: 45 10 45
   :header-rows: 1
