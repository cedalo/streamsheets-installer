TIME
-----------------------------

Calculates the serial number which corresponds to the time specified by given hours, minutes and seconds

**Syntax**

=TIME(Hours, Minutes, Seconds)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Hours
     - A number which specifies hours
   * - Minutes
     - A number which specifies minutes
   * - Seconds
     - A number which specifies seconds

**Return Value**

A serial number value which corresponds to specified time.

**Example**

.. list-table::
   :widths: 45 10 45
   :header-rows: 1

   * - Function
     - Result
     - Comment
   * - =TIME(12, 0, 0)
     - 0.5
     - The given time value represents 12:00 AM.
